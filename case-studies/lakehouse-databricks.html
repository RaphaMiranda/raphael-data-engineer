Modern Data Lakehouse Architecture with Apache Iceberg

Duration Ongoing · 8+ Months
Core Stack
dataset Azure Databricks water_drop Apache Iceberg bolt AWS Glue & EMR

The Challenge

Aged-care and healthcare consulting clients required analytics-ready datasets with strict data governance — while the underlying infrastructure was a fragmented mix of raw S3 dumps, manual SQL, and no lineage tracking. The gap between ingestion and business-ready data caused 48+ hour freshness delays and blocked regulatory compliance reporting.

Legacy Debt

Unstructured S3 dumps with no schema enforcement, duplicated data across zones, and manual transformation scripts with no CI/CD.

Requirements

ACID-compliant near-real-time ingestion, SCD Type 2 history tracking, governed access, and Power BI reporting for healthcare providers.

Medallion Architecture

Systemic Flow: Ingestion → Curated → Aggregated

Strategic Trade-offs

compare_arrows

Apache Iceberg vs. Delta Lake

While Databricks Delta Lake is excellent within a pure Databricks environment, Iceberg was chosen for its open-standard interoperability — enabling Athena, Spark, and Glue to query the same tables without lock-in, critical for a multi-engine AWS workload.

history

SCD Type 2 Design Decision

Healthcare clients require full auditability of record changes over time. SCD Type 2 in the Silver layer preserves complete history, enabling compliance reporting while keeping Gold layer aggregates current and fast.

medallion_pipeline.py 
from pyspark.sql import SparkSession
from pyspark.sql.functions import *

# Silver: SCD Type 2 upsert via Iceberg merge
spark.sql("""
  MERGE INTO silver.patients AS target
  USING bronze.patients_raw AS src
  ON target.patient_id = src.patient_id
    AND target.is_current = true
  WHEN MATCHED AND src.hash != target.hash
    THEN UPDATE SET is_current = false,
                    valid_to   = current_timestamp()
  WHEN NOT MATCHED
    THEN INSERT (patient_id, ..., is_current, valid_from)
         VALUES (src.patient_id, ..., true,
                 current_timestamp())
""")

# Gold: Analytics-ready aggregate
def build_gold_daily_summary():
  return (
    spark.table("silver.patients")
      .filter(col("is_current") == True)
      .groupBy("care_provider", "report_date")
      .agg(count("*").alias("active_patients"))
  )
