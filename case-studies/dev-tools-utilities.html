Editors & IDEs

Containers & Infra

Run anything, anywhere

Docker + Compose

Local orchestration for pipelines, databases, and services during development.

Pulumi

Infrastructure-as-Code using Python and TypeScript for Azure and AWS provisioning with state management.

GitHub Actions

CI/CD pipelines for data pipelines, IaC deployments, and automated testing.

api

API & Database

Query, test, explore

Postman

REST and GraphQL API testing. Collections for Databricks REST API, ADF triggers, and Azure management.

DBeaver

Universal DB client. Connects to Snowflake, PostgreSQL, SQL Server, Synapse, and Hive in one place.

Version Control

Source of truth

Git + GitHub

Branch-based workflows, PR reviews, and protected main branches across all data projects.

Bitbucket

Git hosting for team projects with Jira integration, pipeline triggers, and branch permissions.

pre-commit

Automated hooks for linting, formatting (Black, isort, SQLFluff), and secret scanning before every commit.

draw

Diagramming & Docs

Architecture & communication

Excalidraw

Fast, hand-drawn style architecture diagrams for data flows, lakehouse layers, and infrastructure proposals.

Notion

Project documentation, runbooks, and post-mortems. Single source of truth for technical decisions.

dbdocs.io

DBML-based schema documentation. Publish data model docs as shareable web pages for stakeholders.

monitoring

Monitoring & Observability

Know what's happening

Azure Monitor + Log Analytics

Pipeline health, ADF run alerts, and KQL queries for cost attribution and failure triage.

Great Expectations

Data quality validation with automated expectation suites and HTML data docs generation.

CloudWatch

AWS-native monitoring for Lambda, Glue, and S3 events — log insights, alarms, and cost dashboards.
